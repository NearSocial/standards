# Tags

A list of tags.

## Schema

| Key         | Type         | Description                                                                                                               |
|-------------|--------------|---------------------------------------------------------------------------------------------------------------------------|
| **`[tag]`** | Empty string | A dynamic key represents a tag in the list. For example, `founder` as a key means this tag `founder` belongs to the list. |

## Example

```json
{
  "human-guild": "",
  "founder": "",
  "artist": "",
  "nft": ""
}
```
